Polygon Largest
===============

[geopandas.GeoDataFrame.explode](https://geopandas.org/en/stable/docs/reference/api/geopandas.GeoDataFrame.explode.html) を使って

Geojson のポリゴンを分割して

面積が一番大きいものを選択する。

### Hokkaido

北海道のGeoJson のポリゴンを分割して
面積が一番大きいものを選択する。

### Japan Prefectures

都道府県のGeoJson のポリゴンを分割して
面積が一番大きいものを選択する。
