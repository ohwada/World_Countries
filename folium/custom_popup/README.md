Custom Popup

samples of Custom Popup

- popup_str.py  
display text  
- popup_html.py
display text in red
 - popup_link.py
click to open WEB page  
-popup_img.py
display image  
-popup_iframe.py
click to open a web page in Popup  
- popup_ja.py
display Japanese text  
-popup_ja_width.py
display Japanese text  with max_width  
- popup_vega.py
display the graph
